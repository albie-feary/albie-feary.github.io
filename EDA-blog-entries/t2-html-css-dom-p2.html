What is Grid Based Design?

  • This ubiquitous design leads the eye left to right in row and collumn format. This then creates an intentional user experience.

What is all the hype about responsive web pages?

  • The vast majority of webpages are viewed on portable devices such as phones and tablets, not laptops. In order for best user experience, every element needs to be resized depending on the resolutions of that page.
  • The resolutions are set on % rather than pixels to achieve responsiveness.

What is semantic structure?

  • Semantic structure displays the correct use of commands so that it is displayed as intended.
  • Semantic elements clearly defines an article to the browser or developer.
