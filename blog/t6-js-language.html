The Language of JavaScript

In the JavaScript Language, describe all the use cases of the following:

Parentheses ()

  1. Parentheses are used to execute, invoke or call a function. You can call a function by putting parentheses after an expression that call a function value. These values given to functions are called arguments.
    eg: var win = sheWon()
  2. Parentheses are also used to conatin any parameters, to identify code as a function and create that function.
  3. Parentheses are also used in arithmatic to disrupt the 'precedence' or order of the operators of * / % - and +.
    eg:((x-y)*z)

Brackets []

  1. Brackets can be used to create an array, Eg: var letters = [a,b,c]
  2. Brackets can also be used to access nested properties in an object or array

Braces {}

  1. Braces do for statements what parenthesis do for expressions: they create blocks of code my grouping sequences of statements together to make them count as a single statement.This is often part of a if/else statement or a while loop.
  2. Braces can also be used to create a new Object with literal notation.
    eg: var weather={
    sun: "bright"
    rain: "0%" }
