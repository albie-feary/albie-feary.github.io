Scope in Javascript

Variable scope in Javascript simply means where is this variable visible in the code. What part of the code can use this variable? Below is some code where the variable is declared inside the function.

function simpleFunction() {
var number = 33;
console log(number);
}
simpleFucntion()
//output will be 33

Now, if you were to console log(number); outside of the function, you would get returned the message, undefined. This is because the number variable does not exist outside of the function as far a Javascript is concerned. This is because when var is used inside a fucntion this is what is known as a Local variable. It is only relevant inside the function itself.


If a variable is needed to exist throughout the code you can create what is known as a Global Variable.In this example you would declare var number; outside the function and then inside the function because the variable is already defined, we can write number = 33; and we can call it inside or outside the function.


  • Know that when declaring variables inside a function you may only want to access them inside that function. Otherswise it is available everywhere.
  • Rules of scope only exist in Functions. They do not apply to Objects, if staments, loops and so on.

Nested Functions

As well as Global and Local scoped variables in functions, functions can be created inside other functions. This produces degrees of locality within functions. By hoisting a variable to the top of that program, the variable becomes visible. This variable visitbilty is known as lexical scoping.
