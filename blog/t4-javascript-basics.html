Javascript Basics

How does JavaScript compare to HTML and CSS?

HTML and CSS provide structure and style to a webpage. Javascript is dynamic programming language and allows us to check for the state of things, perform actions based on other conditions, search for things in data structures. Javascript makes web pages interactive.

Control flow and loops

The ability to build logic into a webpage is what makes Javascript so powerfull. To do this, code is written in a way that decisions and repeated tasks are made until certain criteria is met.
For example, if you were writing code to describe someone having breakfast, you could programme it to say that if that person has eggs, they can make a poached egg but if they dont have any eggs then they will have to think of something else.

Accessing data from arrays and objects

An Array in JavaScript is a data type used specifically for storing sequences of values. Accessing and modifying an Array involves using indexes. Objects are similar to arrays but to access the data we use properties. Using Objects we can store data that represents real world objects in a structured way.

Functions

Functions are the reusable blocks of code in Javascript. These JS functions are designed to perform a particular task and are executed when something calls it.
